## 🏁 Getting Started <a name = "getting_started"></a>

Clone

```
git clone https://github.com/raferdev/boardcamp
```

Install Dependencies

```
npm i
```

Start Server

```
npm run dev
```

Use the endpoint.

```
http://localhost:3000
```
